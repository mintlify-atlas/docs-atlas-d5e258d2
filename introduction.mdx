## What is BidCo?

BidCo is a complete auction and raffle management platform designed for real-world bidding scenarios. Whether you're running an open auction where all bids are visible, a blind auction where bids remain secret until the end, or a points-based raffle, BidCo provides the full infrastructure to manage it end-to-end.

The platform is built with a clean client-server architecture:
- **Frontend**: Angular 19+ with standalone components, deployed on Vercel
- **Backend**: Java Spring Boot 3 REST API, deployed on Render
- **Database**: PostgreSQL hosted on Aiven

## Platform Roles

BidCo has three distinct user roles, each with different capabilities:

| Role | Description | Authentication |
|------|-------------|----------------|
| **User (Usuario)** | Registered platform member who can bid, create auctions, and enter raffles | JWT token via `/api/usuarios/login` |
| **Expert (Trabajador)** | Verified specialist with elevated trust, used for expert-managed content | JWT token via expert login |
| **Admin** | Platform administrator who manages raffles and expert accounts | Session-based via `/api/admin/login` |

| Layer | Technology |

| Auth | JWT (jjwt 0.11.5) |
| ORM | Spring Data JPA / Hibernate |
| Object mapping | MapStruct 1.5 |
| Deployment | Vercel (frontend), Render (backend) |
